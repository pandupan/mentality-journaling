# 📔 Mentality Journaling

Aplikasi **jurnal digital** untuk mencatat pemikiran, perasaan, dan refleksi harian. Membantu pengguna melatih mindfulness dan kesehatan mental melalui journaling.

## ✨ Fitur

- **Catatan Harian** — Tulis jurnal harian dengan mudah
- **Mood Tracker** — Pantau suasana hati setiap hari
- **Kategori** — Kelompokkan catatan berdasarkan tema
- **Riwayat** — Lihat perkembangan jurnal dari waktu ke waktu

## 🛠️ Tech Stack

- **Next.js** — React framework
- **TypeScript** — Type safety
- **Tailwind CSS** — Styling

## 🚀 Cara Menjalankan

```bash
npm install
npm run dev
```
